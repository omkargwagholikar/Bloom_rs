## Build and executing
Please make sure you have rust installed.

'''bash
git clone https://github.com/omkargwagholikar/Bloom_rs.git
'''

Install relevant documents 

'''bash
cargo build
'''

Run the tests

'''bash
cargo test
'''

Run bloom code

'''bash
cargo run
'''
